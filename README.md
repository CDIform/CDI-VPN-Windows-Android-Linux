# CDI-VPN-Windows-Android-Linux

ПК версии софта а так же сервер полностью написаны на Rust, Android версия софта полностью написана на Java
Связь с сервером осуществляется по ВебСокетам, трафик в обе стороны шифруется при помощи ChaCha20.

  Установка сервера:

  Для установки серверной части вам потребуется любой VPS на Ubuntu 24.04

  Положите файлы relay_server и install.sh в корневую папку сервера.
  Обновите систему командами sudo apt update && sudo apt upgrade -y

  Запустите скрипт, sudo bash install.sh Скрипт установит сервер, сгенеририует ключи шифрования и выдаст готовый стринг для интеграции в конфиг

Установка завершена успешно

Конфигурация сервера:
{
    "encryption_key": "g~E:g5Q(T)OU$<$pb)u#i5@r!qfwab{,",
    "port": 5921,

========================================
192.168.1.1:5921:g~E:g5Q(T)OU$<$pb)u#i5@r!qfwab{,
========================================


Данную строку нужно вставить в json файл и положить в одну папку к клиентскому софту, примеры json файлов находятся в папках с клиентами. В примере ниже клиент будет подключаться к серверу, имя которого указано в параметре MainRelay
(для виндовс версии дополнительно испльзуется кернел мод драйвер wintun.dll, он необходим для работы)

{
    "MainRelay": "MyServer",
    "Servers": {
      "MyServer": "192.168.1.1:5921:g~E:g5Q(T)OU$<$pb)u#i5@r!qfwab{,",
      "MyServer2": "193.111.201.4:99114:my-super-secret-vpn-key-32bytes!"
    }
  }



В android версии json конфиги заполнять не надо, достаточно просто вставить строку в интерфейсе приложения (RAW). Андроид версия имеет функцию настроек проксирования добавляя в исключения проксирования софт, трафик которого вам не нужно пропускать через ВПН.
